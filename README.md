Un bot Discord semplice, sviluppato con `discord.py`, con comandi base come **!ping**, **!say** e **!info**.

## Comandi

- **`!ping`**: Mostra la latenza del bot.
- **`!say [messaggio]`**: Ripete il messaggio che invii.
- **`!info`**: Mostra informazioni sul bot.

## Prerequisiti

- **Python 3.5+**
- **discord.py**: Installalo con:
  ```bash
  pip install discord.py
  ```

## Setup

1. Crea un bot su [Discord Developer Portal](https://discord.com/developers/applications) e copia il **token**.
2. Sostituisci `IL_TUO_TOKEN` nel file `bot.py` con il tuo token.
3. Esegui il bot con:
   ```bash
   python bot.py
   ```
